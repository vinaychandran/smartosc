## ownego scaffolding by vntrungld

### Structure

```
assets/narita/
--- styles/
--- scripts/
--- fonts/
*.html
```

### Technologies

- Gulp
- Pug
- SASS
- Javascript ES6
- Babel

### Installation

1. Node modules install:
`npm install`

2. Third-party libraries via Bower:
`bower install`

3. Run `gulp serve`

### Production building

Run this to build a production version: `gulp`
